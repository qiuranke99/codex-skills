# Storyboard Prompt

Create a 6-panel storyboard sheet based strictly on the Aurora Cup shot list.

Use the storyboard as sequential shot guidance, not as a static collage. Treat each panel as a separate beat in order.

Each panel should be numbered S01 to S06 and show one clear product-focused beat: hand anticipation, cup reveal, matte surface detail, daily carry motion, warm hero hold, and final clean end frame.

Use minimal text inside panels. Do not render long subtitles, exact package copy, legal claims, QR codes, or detailed logos. Use the product reference image for cup identity. Use the mood reference only for warm morning lighting and tactile clean texture.

Avoid exact reference-video compositions, original brand elements, original copy, and frame-by-frame imitation.
