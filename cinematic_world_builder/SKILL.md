# cinematic_world_builder

## 名称

电影世界生成器

## 作用

将任何用户输入，包括想法、场景、氛围、文化、地点、名称、类型或图像，转化为一个可信、连贯、电影化的想象世界，并输出 9 个不同方面的电影视觉提示。

这个 Skill 不是普通概念艺术生成器，也不是海报提示词生成器。它的目标是让世界像是被摄影机发现的，而不是被设计板展示出来的。输出必须像来自一部未上映电影的静态画面。

默认基础风格：

电影现实主义、胶片颗粒、电影静态画面

除非用户明确指定其他风格，否则保留该基础风格。

## 适用场景

当用户要求以下任务时使用本 Skill：

- 基于一个想法生成电影世界
- 基于一张图扩展世界观
- 基于氛围、文化、地点、名称、类型生成视觉世界
- 生成 9 个电影化世界构建提示
- 为广告、电影、游戏、AI 视频或视觉开发建立世界基础
- 从“角色、动物、建筑、景观、日常、旅行、文化、压力、肖像”等维度探索世界

## 执行步骤

1. 提取用户输入中的核心主题、不可替换元素和显性风格要求。完成标准：能用一句话说清“这个世界不能丢失什么”。
2. 将主题扩展为统一的世界逻辑：地理、气候、材料、身体覆盖、建筑语言、动物关系、技术水平、劳动方式、信仰痕迹和社会压力。完成标准：9 个方面都能共享同一套连续性元素。
3. 按固定顺序生成正好 9 个方面。完成标准：每个方面都有不同叙事目的、世界构建笔记和完整提示。
4. 对照自检清单修正重复、跑题、空泛、海报感、设定图感、居中肖像和缺少可见行动的问题。完成标准：每条提示末尾都有默认负面尾句，且 9 条像同一部电影。

## 输入处理 (Input)

用户输入可能很简单，例如：

- 庆祝
- 火山文明
- 月光下的游牧民族
- 高端香水世界
- 沙漠里的未来寺庙
- 一张参考图

处理规则：

1. 保留用户的核心想法，不要替换主题。
2. 将粗略输入扩展为完整世界。
3. 将氛围转化为地理、气候、文化、材料、仪式、身体覆盖、技术、运动、日常生活和社会压力。
4. 如果输入是图像，需要分析主体、场景、颜色、建筑、身体覆盖、氛围、光线、类型和文化，然后创建一个受其启发的世界。
5. 不要字面复制图像，除非用户明确要求复刻。
6. 不要引入与原始方向无关的文化、技术或时代设定。
7. 世界可以是古代、自然、奇幻、未来、科幻、超现实、精神性、工业化、神话性、后末日或其他方向，由用户输入决定。

## 输出总结构 (Output)

必须输出：

1. 一个诗意的世界名称
2. 2-3 句世界概念
3. 正好 9 个方面
4. 每个方面包含：
   - 方面标题
   - 世界构建笔记
   - 提示

输出格式固定为：

```markdown
[诗意世界名称]

世界概念：
```

## 必须生成的 9 个方面

必须正好生成以下 9 个方面，顺序不得随意改变：

1. 居民
2. 动物
3. 建筑
4. 景观
5. 日常生活
6. 旅行或运动
7. 声音或文化
8. 权力或强度

## 方面定义

### 1. 居民

回答：谁住在这里？

可以是人类、人形、非人类、变异者、神话族群、外星生命、祖先形态、机械生命、类动物生命、类灵体生命或特定物种。

重点展示身份、姿势、身体覆盖、社会角色、肢体语言和具体场景。

禁止：

- 角色表
- 阵列展示
- 时装摆拍
- 物种图鉴
- 居中英雄肖像

### 2. 动物

回答：什么生命与这个世界共存？

可以展示动物或生物正在工作、休息、迁徙、被照料、被狩猎、被崇拜、被恐惧、被骑乘、被跟随，或通过足迹、阴影、呼吸、骨头、巢穴、皮毛、声音、运动痕迹间接出现。

禁止：

- 生物设定表
- 标本展示
- 博物馆式陈列
- 没有环境关系的怪物图

### 3. 建筑

回答：他们在哪里庇护、聚集、统治、隐藏或记忆？

必须展示正在被使用的建筑空间，而不是空荡荡的建筑设计图。

可以体现：

- 庇护
- 聚集
- 统治
- 隐藏
- 修复
- 祈祷
- 烹饪
- 睡眠
- 守望
- 通道
- 烟雾
- 损坏
- 工具
- 足迹
- 织物
- 光线
- 仪式

禁止：

- 建筑效果图
- 空场景
- 对称宣传片机位
- 纯设计展示

### 4. 景观

回答：什么土地塑造了这个文明？

地理、气候和地形必须主导画面。通过远处身影、动物路径、废墟、作物线、烟雾、足迹、船只、旗帜、侵蚀、天气或信仰痕迹展示尺度。

重点是环境作为力量，而不是旅行风景照。

禁止：

- 普通风光摄影
- 空泛奇观
- 没有文明痕迹的背景图

### 5. 日常生活

回答：这里什么普通行动反复发生？

展示日常劳动、食物、手工艺、洗涤、修复、贸易、家庭节奏、训练、玩耍、火护理、工具准备或家务重复。

保持普通，不要变成仪式、战斗、灾难或象征性肖像。

禁止：

- 过度戏剧化
- 战斗场面
- 盛大仪式
- 灾难场面
- 纯摆拍

### 6. 旅行或运动

回答：存在者如何在地方之间移动？

通过距离、地形、门槛或阻力展示运动。

可以是：

- 行走
- 骑乘
- 划船
- 拖拽
- 攀爬
- 漂流
- 迁徙
- 离开
- 到达
- 路线
- 交通
- 天气中的移动
- 负重
- 节奏和努力

禁止：

- 纯风景
- 无行动的人物站立
- 普通交通工具展示

### 7. 声音或文化

回答：这个世界听起来像什么、记住什么或表演什么？

通过视觉让声音或意义可见。

可以展示：

- 吟唱
- 聆听
- 乐器
- 钟声
- 工具敲击材料
- 舞蹈
- 风中物体
- 沉默
- 口述传统
- 仪式歌曲
- 哀悼
- 表演
- 回声空间

禁止：

- 抽象文化介绍
- 文字说明牌
- 舞台海报感
- 没有声音来源的空场景

### 8. 权力或强度

回答：什么压力考验这个世界？

展示力量、危险、权威、速度、冲突、耐力或张力。

可以是：

- 风暴
- 洪水
- 仪式权威
- 社会沉默
- 危险劳动
- 危机
- 惩罚
- 热量
- 稀缺
- 追逐
- 对峙
- 变化前的瞬间

禁止：

- 普通战争大片
- 无原因的爆炸
- 夸张英雄动作
- 脱离世界逻辑的灾难

### 9. 肖像

回答：什么私人时刻承载这个世界的灵魂？

展示一个亲密、私人的感情时刻。主体可以是人类或非人类。

必须避免居中肖像构图。

可以展示：

- 反应
- 手势
- 瞥视
- 伤口
- 呼吸
- 脏手
- 半光脸
- 隐藏表情
- 画面外聆听
- 与动物、物件、亲人、遗物或环境的细微关系

禁止：

- 居中大头照
- 时装肖像
- 角色设定图
- 证件照式构图

## 方面分离规则

每个方面必须有不同的叙事目的，不得重复场景类型。

区分如下：

- 日常生活 = 普通重复行为
- 旅行或运动 = 地点之间的移动
- 声音或文化 = 声音、记忆、表演、意义
- 权力或强度 = 压力、危机、权威、危险、冲突
- 肖像 = 私人反应或亲密瞬间
- 景观 = 土地作为塑造文明的力量
- 建筑 = 正在被使用的建筑空间
- 动物 = 与文明共存的其他生命
- 居民 = 居住者身份、身体覆盖、角色和社会存在

如果两个方面开始相似，必须主动改写其中一个。

## 世界连续性规则

所有 9 个方面必须像同一部电影、同一个世界。

必须通过以下元素维持连续性：

- 气候
- 地理
- 材料
- 身体覆盖
- 建筑语言
- 工具
- 符号
- 动物
- 行为
- 地形
- 光线
- 颜色逻辑
- 纹理
- 劳动方式
- 信仰痕迹
- 技术水平

禁止：

- 每个提示像不同世界
- 随机切换时代
- 随机切换文化
- 引入无关技术
- 忽然现代化，除非用户输入要求
- 过度混搭导致世界失去可信性

## 场景和相机规则

每个方面必须回答：

“这个画面中正在发生什么？”

每个提示都必须像一个被摄影机捕捉到的电影时刻，有：

- 前景
- 中景
- 背景
- 行动
- 前后感
- 空间关系
- 观察性相机

相机应该像发现这个时刻，而不是为了展示世界而摆布场景。

必须变化：

- 景别
- 机位高度
- 镜头距离
- 构图阻挡
- 前景遮挡
- 负空间
- 景深
- 运动痕迹
- 观看角度

但变化必须服务于故事，不得随机。

可使用以下摄影逻辑：

- 低角度、地面级构图、前景尺度或微小身影：表现巨人、巨型动物、古老结构、威胁、敬畏或神圣尺度。
- 高角度、俯视、负空间：表现隔离、仪式秩序、迁徙、地理或社会秩序。
- 近距离、局部脸、手、呼吸、磨损工具、浅景深：表现亲密、劳动、情感或张力。
- 肩后、门框、障碍物、画面中画面：表现秘密、发现、等级或见证。
- 深度、剪影、反射、烟雾、尘土、雨、火光、部分可见性：表现旅行、压力、神秘、危险、记忆或精神重量。

避免 9 个方面都变成中远景宽画面。

- 工具
- 姿势
- 劳动
- 气候
- 技术

- 动物关系
- 身体覆盖
- 建筑使用方式

- 环境痕迹

## 写作规则

输出必须视觉化、简洁、可直接用于图像生成。

不要写长历史、时间线、百科解释、政治制度、物种分类或抽象背景，除非用户明确要求。

世界构建笔记必须是一句简短视觉句子。

每个提示应写成一个干净完整的段落，重点包含：

- 主体位置
- 环境
- 身体覆盖
- 材料
- 光线
- 天气
- 相机距离
- 构图
- 画面中正在发生的动作
- 电影现实主义

避免空泛词，例如：

- 史诗
- 美丽
- 酷
- 震撼
- 高级
- 梦幻

除非这些词由具体视觉细节支撑。

禁止使用：

- 在某某在世艺术家风格中
- 在某某在世导演风格中
- 某某工作室风格
- 海报
- 角色表
- 设定集
- 概念艺术板
- 产品目录
- 阵列图
- 纯背景
- 纯肖像
