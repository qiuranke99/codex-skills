# Quality Rubric

Use this rubric after the structural audit. A commercial-ready output must pass every gate.

## Hard Gates

| Gate | Pass Standard | Fail Pattern |
| --- | --- | --- |
| Count | Exactly 10 numbered prompts | 9, 11, bullets, merged items |
| Format | Title, composition line, prompt label, prompt paragraph | Missing label or inconsistent structure |
| Core theme | Same subject and situation across all prompts | Ten unrelated scenarios |
| Composition diversity | Ten distinct camera/composition concepts | Repeated low angle, repeated close-up, cosmetic variation |
| Film-still quality | Reads like a captured scene moment | Poster, portrait, concept art, beauty shot |
| Spatial depth | Foreground, midground, and background are implied or explicit | Flat subject-only framing |
| Light source | Practical or natural source is named or visually implied | Generic "cinematic lighting" only |
| Environment texture | Dust, weather, reflection, haze, surface, shadow, grain, or motion texture | Sterile digital image request |
| Narrative pressure | Action, emotion, tension, uncertainty, or transition | Static pose without story |
| Negative phrase | Default negative appended to every prompt | Missing, translated inconsistently, or only once |

## Scoring

Score each prompt 0-2 on:

- composition specificity
- spatial layering
- light logic
- environmental texture
- narrative tension
- model usability

Commercial-ready threshold: every prompt scores at least 9/12, and no prompt scores 0 on any category.

## Revision Rules

- If a prompt is beautiful but vague, replace adjectives with camera and scene evidence.
- If two prompts share the same camera idea, change one at the composition-family level.
- If the subject becomes a different story, restore the locked core scene.
- If the prompt looks like a poster, add scene action, imperfect framing, foreground interruption, or practical light.
- If the image model would need more direction, add camera height, lens/framing implication, foreground, midground, background, and light source.
